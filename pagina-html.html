HTML5

HTML5 es un lenguaje de estiquetas significa (Hypertex Markup lenguaje)

y se usa para hacer la extructura de la pagina web


Etiqueta principales

!DOCTYPE

Etiqueta que se usa para indicar que la pagina esta realizada en la version de HTML

HTML

Dentro de estas etiquetas se debe poner toda la pagina web


Etiquetas Principales dentro de HTML

head

Head es la cabeza de la pagina web,aqui agregamos el titulo de la pagina ( pestaña del navegador),

enlaces de css, enlaces de jvascript, Etiquetaas Metay demas enlaces

Body

El body es el cuerpo de la pagina web, Aqui se agrega todos los componentes de la pagina.

Footer

E Footer es los pies de la pagina web,en el ponemos enlaces a redes sociales,numeros telefonicos,Direcciones,etc


Contenedores

Nav (Menu)

el nav o el emnu es donde ponemos los enlaces de navegacion de la pagina web

header ( Cabesera)

el header( cabesera)es donde podemos poner el menu y/o la portada de la pagina web

Aside (A un lado)

Este contenedor suele estar a lso lados,izquierda o derecha. Y se suele usar para menus alternos o publicidad

Section (secciones)

Este contenedor se usa para meter secciones de contenido

Article (Articulos)

Los articulos son pequeños contenedores que se suelen usar dentro de otros contenedores y se usan

para hacer pequeños articulos o cuadros con contenidos

DIV (Son diviciones)- No recomendable usar.

Antiguamente se solia usar peero ya cayo en des-uso, Ya que los dispositivis moviles

no solian reconocer que parte de la pagina era.


Etiquetas HTML5

Etiquetas H

Las etiquetas H van desde el valor 1 hasta el valor6.

El valor menor tiene mas importacia que el valor mayor

El valor menor tiene mas tamaño que el valor meyor.

h1 h2 h3 h4 h5 h6



p (PARRAFOS)

las o son etiquetas que se usan para poner parrafos


A ( enlaces)

ls etiqueta a se usa para crear enlaces (se puede usar en cualquier parte)

la propiedad o atributo que enlaza es el href=""

llamammos a paginas con su nombre ( Pagina.html) o a secciones con el # (#seccion1)


IMG () MAGENES

La etirqueta img se usa para enlazar imagenes

la propiedad o articulo que enlaza es src=""


UL (agrupar listas)

la etiqueta ul se usa para agrupar li( lista)


LI (Listas)

se usa para hacer listas


BUTTON ( Boton)

La etiqyeta button se usa para hacer botones
